# Video Conference App

## Description
A simple video conferencing application built using React, Node.js, Socket.io, and WebRTC.

## Features
- Video Calling
- Audio Support
- Real-time Chat
- Screen Sharing
- Responsive Interface

## Technologies Used
- React.js
- Node.js
- Express.js
- Socket.io
- WebRTC

## Installation

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm start
